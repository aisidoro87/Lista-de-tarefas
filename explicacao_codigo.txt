### 1. Seleção de Elementos do DOM
O código começa selecionando os elementos HTML que vamos manipular.

```javascript
const taskInput = document.getElementById('taskInput');
const addBtn = document.getElementById('addBtn');
const taskList = document.getElementById('taskList');
const completedTaskList = document.getElementById('completedTaskList');
const errorMessage = document.getElementById('errorMessage');
const clearCompletedBtn = document.getElementById('clearCompletedBtn');
```
*   **O que faz:** Cria referências (variáveis constantes) para os elementos da sua página HTML usando o `id` deles.
*   **Por que:** Para que o JavaScript possa "ler" o que foi digitado (`taskInput`), "ouvir" cliques nos botões (`addBtn`, `clearCompletedBtn`) e "escrever" novas tarefas nas listas (`taskList`, `completedTaskList`).

---

### 2. A Função `addTask()`
Esta é a função principal. Ela é executada toda vez que o usuário tenta adicionar uma nova tarefa.

#### Validação de Entrada
```javascript
function addTask() {
    const text = taskInput.value.trim();
    if (text === '') {
        errorMessage.textContent = 'Por favor, digite uma tarefa.';
        return;
    }
    errorMessage.textContent = ''; // Limpa a mensagem de erro se houver
```
*   **`taskInput.value.trim()`:** Pega o texto digitado e remove espaços em branco do começo e do fim.
*   **`if (text === '')`:** Verifica se o campo está vazio. Se estiver, exibe a mensagem de erro no elemento `<p>` e usa `return` para parar a função imediatamente (não cria a tarefa).
*   **`errorMessage.textContent = ''`:** Se passou pelo teste (tem texto), limpa qualquer mensagem de erro anterior.

#### Criação da Estrutura Básica
```javascript
    const li = document.createElement('li');
    
    const span = document.createElement('span');
    span.className = 'task-text';
    span.textContent = text;
    
    const divActions = document.createElement('div');
    divActions.className = 'actions';
```
*   **O que faz:** Cria os elementos HTML na memória do navegador (ainda não visíveis na página).
*   **Estrutura:** Cria um `<li>` (o item da lista), um `<span>` (para guardar o texto da tarefa) e uma `<div>` (para agrupar os botões de ação).

#### Botão de Editar
```javascript
    // Botão de Editar
    const editBtn = document.createElement('button');
    // ... (configuração de classe e ícone)
    editBtn.onclick = () => {
        if (li.classList.contains('editing')) {
            // Lógica de SALVAR a edição
            const input = li.querySelector('.edit-input');
            const newText = input.value.trim();
            
            if (newText !== '') {
                span.textContent = newText; // Atualiza o texto
                input.remove(); // Remove o campo de input
                span.style.display = ''; // Mostra o texto novamente
                editBtn.innerHTML = '&#9998;'; // Volta ícone para lápis
                li.classList.remove('editing');
                // Mostra os outros botões novamente
                completeBtn.style.display = '';
                deleteBtn.style.display = '';
            }
        } else {
            // Lógica de INICIAR a edição
            span.style.display = 'none'; // Esconde o texto
            completeBtn.style.display = 'none'; // Esconde botão concluir
            deleteBtn.style.display = 'none';   // Esconde botão excluir
            
            const input = document.createElement('input'); // Cria input
            input.type = 'text';
            input.className = 'edit-input';
            input.value = span.textContent; // Põe o texto atual no input
            li.insertBefore(input, divActions);
            input.focus(); // Foca no input
            editBtn.innerHTML = '&#10004;'; // Muda ícone para check
            li.classList.add('editing'); // Marca que está editando
        }
    };
```
*   **Lógica:** Este botão funciona como um interruptor (toggle).
    *   Se a classe `editing` existe: Ele pega o valor do input, salva no span e restaura a visualização normal.
    *   Se a classe `editing` **não** existe: Ele esconde o texto e os outros botões, cria um campo de input no lugar do texto para você digitar e muda o ícone do botão para "salvar".

#### Botão de Concluir
```javascript
    // Botão de Tarefa Concluída
    const completeBtn = document.createElement('button');
    // ...
    completeBtn.onclick = () => {
        li.classList.add('completed-task'); // Adiciona estilo riscado
        divActions.remove(); // Remove os botões originais (editar/excluir/concluir)

        // Cria nova área de ações para a tarefa concluída
        const completedActionsDiv = document.createElement('div');
        completedActionsDiv.className = 'actions';

        // Cria a data
        const completionDate = document.createElement('small');
        // ... (formata a data e hora atuais)

        // Cria botão de excluir específico para tarefa concluída
        const singleDeleteBtn = document.createElement('button');
        // ...
        singleDeleteBtn.onclick = () => {
            if (confirm('Tem certeza que deseja excluir esta tarefa concluída?')) {
                li.remove();
            }
        };

        // Monta a nova estrutura e move para a outra lista
        completedActionsDiv.appendChild(completionDate);
        completedActionsDiv.appendChild(singleDeleteBtn);
        li.appendChild(completedActionsDiv);

        completedTaskList.appendChild(li); // Move do taskList para completedTaskList
    };
```
*   **O que faz:** Transforma a tarefa pendente em concluída.
*   **Detalhe importante:** Ele **remove** os botões antigos (`divActions.remove()`) e cria novos elementos (Data e Botão de Excluir simples) antes de mover o elemento `<li>` inteiro para a lista de baixo (`completedTaskList`).

#### Botão de Excluir (Tarefas Pendentes)
```javascript
    // Botão de Excluir
    const deleteBtn = document.createElement('button');
    // ...
    deleteBtn.onclick = () => {
        // Esconde todo o conteúdo atual do LI
        const children = Array.from(li.children);
        children.forEach(child => child.style.display = 'none');

        // Cria a caixa de confirmação
        const confirmDiv = document.createElement('div');
        confirmDiv.className = 'confirmation-box';
        confirmDiv.innerHTML = `...html da pergunta Sim/Não...`;
        li.appendChild(confirmDiv);

        // Lógica do SIM: remove o LI da tela
        confirmDiv.querySelector('.btn-yes').onclick = () => li.remove();
        
        // Lógica do NÃO: remove a pergunta e mostra o conteúdo de volta
        confirmDiv.querySelector('.btn-no').onclick = () => {
            confirmDiv.remove();
            children.forEach(child => child.style.display = '');
            // Se estava editando, mantém o texto escondido (pois o input está lá)
            if (li.classList.contains('editing')) span.style.display = 'none';
        };
    };
```
*   **Segurança:** Em vez de usar `confirm()` do navegador (janela pop-up), ele cria uma interface HTML dentro da própria tarefa perguntando se você tem certeza. Ele esconde temporariamente o conteúdo da tarefa para mostrar essa pergunta.

#### Montagem Final e Limpeza
```javascript
    divActions.appendChild(editBtn);
    divActions.appendChild(completeBtn);
    divActions.appendChild(deleteBtn);
    li.appendChild(span);
    li.appendChild(divActions);
    
    taskList.appendChild(li); // Adiciona o LI pronto na lista UL
    taskInput.value = ''; // Limpa o campo de digitação principal
    taskInput.focus(); // Coloca o cursor de volta no campo
}
```
*   **O que faz:** "Cola" os botões dentro da `divActions`, cola o texto e a `divActions` dentro do `li`, e finalmente coloca o `li` dentro da lista visível na tela (`taskList`).

### 3. Event Listeners (Interatividade)

```javascript
addBtn.addEventListener('click', addTask);
```
*   Quando clica no botão "Adicionar", roda a função `addTask`.

```javascript
taskInput.addEventListener('keypress', (e) => { 
    if(e.key === 'Enter') addTask(); 
});
```
*   Quando aperta uma tecla dentro do campo de texto, verifica se é "Enter". Se for, roda `addTask`.

```javascript
taskInput.addEventListener('input', () => {
    if (errorMessage.textContent) errorMessage.textContent = '';
});
```
*   Assim que você começa a digitar algo novo, a mensagem de erro vermelha ("Por favor, digite...") desaparece.

```javascript
clearCompletedBtn.addEventListener('click', () => {
    if (completedTaskList.children.length > 0 && confirm('Tem certeza...?')) {
        completedTaskList.innerHTML = '';
    }
});
```
*   Verifica se tem itens na lista de concluídos. Se tiver, pede confirmação. Se o usuário aceitar, `innerHTML = ''` apaga tudo o que está dentro da lista de concluídos.
